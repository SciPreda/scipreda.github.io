How To Use The SciPred Calculator

About The Calculator

The SciPred Calculator, also called the SCPDCALC, is a slightly experimental project that includes a variety of things you want to calculate, graphs and calculus excluded.

It can range from calculating the circumference of a circle to the total perimeter of an icosahedron to the direction of the parabola formed by a quadratic function.

You are given a minimum of 1 input to fill in, and a maximum of 4 inputs to fill in. The number of inputs and the way you want to process those inputs is up to you.

The amount of possibilities for processing inputs varies for how many inputs you have selected.

Programmer's Warnings

Some outputs may return NaN, which is an acronym for "Not A Number". This may mean that the resulting output is an imaginary number (most likely from using the Math.sqrt function to determine the square roots of negative numbers), or the answer is Undefined.

Numbers with a considerable amount of digits will be shortened into Scientific Notation. These are numbers with the lowercase letter e in them.

Sometimes, when handling decimals (usually in square roots or trigonometric functions), the answer will be inaccurate by a small amount (e.g. 5.33 + 5.2 = 10.530000000000001). These are called floating-point errors, and in most cases it is best to just round the results off to the hundredth digit.

The order of inputs is extremely important. You might accidentally mistake the input for the base of a triangle as an input for the height of a triangle. In some cases the outputs may return NaN even though there is nothing wrong with the values you have inputted. In this case, you may want to consider reading the Input Guide below.

Leaving an input empty will default the value of that input to zero.

Input Guide

Input values here from left to right belong to their respective inputs from top to bottom. Make sure that the resulting inputs for geometric calculators do not lead to impossible geometrical formations.

For calculations that require 1 input:

  • Circle: radius
  • Cube: side
  • Decimals: decimal
  • Dodecahedron: side
  • Equilateral Triangle: side
  • Exponents and Roots: number
  • Icosahedron: side
  • Increments and Decrements: number
  • Multipliers and Divisions: number
  • Octahedron: side
  • Properties: number
  • Sphere: radius
  • Square: side
  • Tetrahedron: side

For calculations that require 2 inputs:

  • Basic Arithmetic: number, number
  • Cone: height, radius (circular base)
  • Cylinder: height, radius (circular base)
  • Ellipse: distance of vertex, distance of co-vertex
  • Exponents and Roots: number, number
  • Linear Equation: a, b
  • Linear Function: a, b
  • Parallelogram: altitude, base
  • Point on a Plane: x, y
  • Rectangle: length, width
  • Regular Polygon: number of sides, side length
  • Right Triangle: adjacent, opposite
  • Triangle: height, base

For calculations that require 3 inputs:

  • Circle: radius, x, y
  • Cuboid: length, width, height
  • Parallelogram: altitude, base, angle
  • Point on a Plane: x, y, z
  • Quadratic Equation: a, b, c
  • Quadratic Function: a, b, c
  • Trapezoid: smaller base, larger base, height
  • Triangle: side, side, side

For calculations that require 4 inputs:

  • Cyclic Quadrilateral: side, side, side, side
  • Trapezium: larger base, left leg, smaller base, right leg
